## 🐛 Troubleshooting

### Common Issues

**Problem**: Receipt is too wide  
**Solution**: Set printer paper size to 58mm

**Problem**: Text is cut off at edges  
**Solution**: Reduce print scale to 95%

**Problem**: Font doesn't look monospace  
**Solution**: Enable "Print background graphics" in browser

**Problem**: Discount not showing  
**Solution**: Discount only shows when `bill.discount > 0`

**Problem**: Receptionist shows "N/A"  
**Solution**: Ensure bill has `created_by_name` or `receptionist` field

### More Help

See: `THERMAL_RECEIPT_GUIDE.md` → Troubleshooting section
